Block element

ব্লক এলিমেন্ট অনেকটা বড় খালুর মতো। যেমন বড় খালু আমাদের বাসায় আসলে আমরা একটি ঘর এবং ঘাট সম্পূর্ণ ছেড়ে দেই এবং তাকে সম্মানের সাথে আরাম মত একা থাকতে দেই, ব্লক এলিমেন্ট ও ঠিক তেমনি ডিক্লেয়ার করার সাথে সাথেই নিজস্ব অবস্থান থেকে সম্পূর্ণ weidth নিয়ে আরাম মত বসে পড়ে। এরা বড়, তাই সুবিধামতো এদের hight, Width এবং Margin, Padding দেয়া যায় ৷ যেমন h1 একটি ব্লক এলিমেন্ট ৷ নিচে তাকে ডিক্লেয়ার করে তার ব্যাকগ্রাউন্ড সেট করে দিলে আমরা তার attitude দেখতে পারবো।

আমি h1 ট্যাগ। আমার আরেকটা পরিচয় হলো আমি ক্লক এলিমেন্ট।

Inline element

মনে কর, তুমি নিজেই inline element। বড় খালু বাসায় আসলে তুমি রুমটা তাকে ছেড়ে দিয়েছো ঠিকই। কিন্তু তোমাকেও তো কোথাও না কোথাও থাকতে হবে। ওইদিকে আরো অনেক আত্মীয়-স্বজন বাকি সব রুম দখল করে নিয়েছে। তারাও সিনিয়র। তাই তাদেরকেও কোথাও সড়িয়ে দেওয়ার সাহস তোমার নেই। এমন সিচুয়েশনে তুমি নিশ্চয় তোমার সমবয়সী আর ছোটো ভাইদেরকে নিয়ে কোনো এক রুমের ফ্লোরে গাদাগাদি করে থাকবে, ইনলাইন এলিমেন্টও ঠিক এমনি, এরা সম্পূর্ণ জায়গা দখল করে না। বরং, এরা সবাই মিলে এক লাইনে গাদাগাদি করে থাকে। এদের ডানে বামে Margin, এবং সব দিকে Padding দেয়া যায়, কিন্তু hight, Width দেওয়া যায়না। তবে এদের css প্রোপার্টি display: block; করে দিলে এরাও ব্লক এলিমেন্ট এর মত সুবিধা পায়। অনেকটা জেদ করে অধিকার আদায় করে নেওয়ার মতো বেপার। ছোটো তো, তাই জেদ করলেই কাজ হয়ে যায়।

Flexbox সমগ্র
Flexbox মাধ্যমে আইটেমগুলকে পাশাপাশি আনার উপায়

Flexbox আমাদের কোনো আইটেমকে পাশাপাশি আনতে সাহায্য করে। এক্ষেত্রে একটি মা এলিমেন্ট এর কোলের মধ্যে কিছু বাচ্চাকাচ্চা রাখা হয় এবং বাচ্চাগুলোকে পাশাপাশি আনতে মা এর ক্লাস নেম ধরে display: flex; করে দেওয়া হয়।

Flexbox দিয়ে ৩ টি div কে পাশাপাশি এনে দেখানো হলো

আমি ১ম সন্তান।

আমি ২য় সন্তান।

আমি ৩য় সন্তান।

এদের মা div এর ওপর display: flex; ব্যবহার করে পাশাপাশি আনা হয়েছে। এবং flex box এর আর একটি প্রোপার্টি justify-content: space-evenly; ব্যবহার করে তাদের সবার পাশে সমপরিমান জায়গা দেওয়া হয়েছ। এমন আরো কিছু স্পেসিং ভ্যালু রয়েছে। যেমনঃ space-between, space-around, space-evenly. আকাশি রং এর div টি মূলত মা div, যার কোলের মধ্যে ৩টি সন্তান রয়েছে।

Flexbox এর মাধ্যমে কোনো আইটেমকে মাঝামাঝি আনার উপায়

কোনো সন্তান div (Child div) কে মা div (Mother div) এর একদম মাঝামাঝি আনতে মা div এর মধ্যে দুইটি Property ব্যবহার করা হয়। একটি justify-content: center; আরেকটি align-items: center; এই দুইটি Proparty ব্যবহার করলে সন্তান গুলো মা এর height এবং width অনুযায়ী একদম মাঝখানে চলে আসবে। এখন নিশ্চয় মনে একটা প্রশ্ন ধাক্কা মারতেছে। প্রশ্নটা হলো, কে কোনদিকে কাজ করে তা তো বললেন না?! হ্যা এইটা এখনো বলিনি কারন flexbox কিন্তু horizontal বা virtical হিসেবে চিন্তা করে না। flexbox দুইটা axis হিসেবে চিন্তা করে। একটা Main axis অন্যটা হলো Cross axis, যদি আমরা justify-content: center ব্যবহার করি তাহলে তা Main axis বরাবর মাঝখানে চলে আসে। আর যদি আমরা align-items: center; ব্যবহার করি তাহলে তা Cross axis বরাবর মাঝখানে চলে আসে। এইভাবে চিন্তা করার পেছনে একটা কারণ হলো, যখন আমরা flex-diretion: column; ব্যবহার করব, তখন Main axis আর Cross axis কলাম এর সাপেক্ষে ঘুরে যাবে। যার ফলে justify-content আর align-items প্রোপারটি দুটিও বিপরীমূখী আচরন করবে। আমি শুধু ফটর-ফটর করেই যাচ্ছি। চলেন প্রাক্টিক্যাল দুইটা উদাহরণ দেখা যাক।

Flexbox এর Child এর মাঝে জায়গা বন্টন করার উপায়

আমরা চাইলে Flexbox এর বাচ্চা আইটেমগুলোর মাঝে আনুপাতিক হারে জায়গা ভাগ করে দিতে পারি। এক্ষেত্রে, Child আইটেমগুলোর প্রোপার্টি হিসেবে flex-grow ব্যবহার করতে হবে। flex-grow এর ডিফল্ট ভ্যাল্যু 0 থাকে। এর ভ্যালু প্রতিটি Child আইটেম এর মধ্যে 1,2,3 ইত্যাদি দেওয়া হয়। আর এই অনুপাত হিসেবেই মা এর কোলে থাকা বাচ্চা গুলোকে জায়গা ভাগ করে দেওয়া হয়।

CSS Grid এর সমগ্র
CSS Grid ব্যাবহারের নিয়ম

CSS Grid এর ক্ষেত্রেও একটি মা এলিমেন্ট এর মধ্যে কিছু বাচ্চা-কাচ্চাদের রাখতে হবে। তারপর মা এলিমেন্ট এর ওপর display:grid; প্রোপার্টি ব্যাবহার করতে হবে। এখনো কোনো কিছুই হয়নি। কাজ আর অল্প একটু বাকি আছে। CSS Grid হলো 2 dimensional layout তৈরি করতে পারে। তাই, display:grid; ভ্যালু সেট করার পর আমরা কতোগুলো column বা row চাই, তা বলে দিতে হবে। নিচে CSS grid এর একটি উদাহরণ দেওয়া দেওয়া হয়েছে।

ওপরে একটি মা div (class="grid-container") এর মধ্যে ১২ টি সন্তান div (class="grid-item") রাখা হয়েছে। এর পর মা div এর ভেতর display:grid; প্রোপার্টি বসিয়ে, আমরা কয়টা column চাই তা বলে দেওয়া হয়েছে এবং বাচ্চাগুলোর প্রত্যেকের মাঝে কতটুকু Gap চাই তা বলে দেওয়া হয়েছে। এখানে, মা div এর মধ্যে ব্যবহৃত CSS প্রোপার্টিগুলো নিচে দেখানো হলোঃ


.grid-container {

display: grid; >>> তোমার বাচ্চাগুলোকে গ্রিড হিসেবে সাজাও।
grid-template-columns: repeat(4, 1fr); >>> তোমার বাচ্চাগুলোকে ৪ কলামে বিভক্ত করো এবং সবাইকে সমপরিমান জায়গা দাও।
grid-gap: 20px; >>> প্রতিটি বাচ্চার মধ্যবর্তী দূরুত্ব ২০ পিক্সেল করে দাও।
}

এখানে fr অর্থ হলো fraction. এটি মা div এর অভ্যন্তরে অবস্থিত বাচ্চা div গুলোকে আনুপাতিক হারে যায়গা বন্টন করে দেয়। আর একটু উদাহরণ দিয়ে একটি ছোট্ট কিন্তু গুরুত্বপূর্ন বিষয় যেনে নেই।
grid-template-columns: repeat(4, 1fr) এই প্রোপার্টিতে repeat(4, 1fr) ফাংশন দ্বারা বোঝানো হয়েহে, তুমি ৪ টা কলাম বানাও এবং প্রতিটি কলামকে 1 fraction করে জায়গা দাও। যার ফলে আমরা সবগুলো কলামকে সমান দেখতে পাচ্ছি। একই ভাল্যু আমরা অন্যভাবেও লিখতে পারি।
grid-template-columns: 1fr 1fr 1fr 1fr;. তাহলেও একই কাজ হবে।
grid-column আর grid-row বর্ধিত করার উপাই

visibility: hidden; আর display: none; নিয়ে যত বিভ্রান্তি
visibility: hidden;

মনে করো ৩ টা div এর মধ্যে মাঝখানের div টা তোমার বাড়ি। এখন যদি তোমার বাড়ীর ওপর visibility: hidden; প্রয়োগ করো তাহলে দেখতে পারবা তোমার বাড়ির জায়গা আছে কিন্তু বাড়ি নাই। মানে তুমি এখানে একটা গ্রেনেট মেরে দিয়েছো।
তবে, জায়গাটা ইন্সপ্যাক্ট করে দেখতে পারবে এখানে কোনো একদিন তোমার বাড়ি, জায়গা সব ছিলো। পারমানবিক বোম সব শেয করে দিয়েছে।

Position Relative VS absolute আর z-index নিয়ে কিছু কথা
position: relative;

position relative এর অর্থ হলো, যে আশ্রয় দেবে। position: relative; প্রয়োগ করলে কোনো একটা absolute আইটেম তাকে কেন্দ্র করে অবস্থান পরিবর্তন করে।

position: absolute;

position absolute এর অর্থ হলো, যে আশ্রয় নেবে। position: absolute; প্রয়োগ করলে সেই আইটেম relative আইটেমকে কেন্দ্র করে তার অবস্থান পরিবর্তন করে।

position: relative; আর position: absolute; এর ছোট্ট উদাহরণঃ

position: relative; = সূর্য্য
position: absolute; = গ্রহ

position: relative; (সূর্য্য)
position: absolute; (গ্রহ)

অপরের উদাহরণে সবুজ রঙের বাক্সকে position: relative; করা হয়েছে। আর লাল বাক্সকে বলা হয়েছে তোমার position: absolute;। তুমি relative সাপেক্ষে ডান পাশ থেকে 40px এবং নিচ থেকে 10px জায়গা ছাড়ো। তাই, (position: absolute;) গ্রহ আমাদের সুর্য্য (position: relative;) এর সাপেক্ষে ডানে 40px এবং নিচ থেকে 10px জায়গা ছেড়েছে।

z-index জিনিসটা কি?

Overflow নিয়ে কিছু কথা
overflow: hidden; আর overflow: scroll; কি এবং কেনো?

বেশি রোমান্টিক mode এ গিয়ে কেউ যদি প্রিয়তমার নিকট খুব বেশি ভালোবাসার বহিপ্রকাশ করে থাকে, তখন মানুষ বলে ভালোবাসা যেন উতলাইয়া পরতেছে। এই উতলাইয়া পরার বিষয়টাই হলো Overflow. যদি কখনো কোনো div এর height এর তুলনায় তার মধ্যের text গুলো বেশি হয়ে যায়, তখন লেখাগুলো div এর বাহিরে চলে যায়। overflow প্রোপার্টি ব্যবহার করে তা নিয়ন্ত্রন করা হয়।

(CSS spacificity) selector এর শক্তি পরিক্ষা
CSS spacificity কি?

CSS ব্যবহারের সময় আমরা বিভিন্ন ধরনে selector এর সাহায্য নিয়ে থাকি। কখনো আমরা tag name ব্যবহার করে থাকি, কখনো class name কখনো বা id এর সাহায্যে কোনো html এলিমেন্টকে select করে তার ওপরে স্টাইল দিয়ে থাকি। আমাদের জানা দরকার এই seletor গুলোর মধ্যে কার আগে কাকে মূল্যয়ন করা হয়। এদের গুরুত্ব বোঝানোর জন্য আমি একটা সহজ উদাহরণ তুলে ধরলাম।

CSS spacificity এর উদাহরণ

#id= মন্ত্রী
.class= এম. পি.
tagname= উপজেলা চেয়ারম্যান

উপরের উদাহরণ থেকে বুঝে নাও কোনো এলিমেন্ট select করার জন্য কাকে আগে প্রাধান্য দেওয়া হয়। এছাড়া আরো কিছু চ্যালা-পান্ডা আছে। চাইলে গুগোল করে দেখে নিতে পারো।

Pseudo Class এর ধারনা
Pseudo Class কি?

অনেক সময়ে web page এর মধ্যে বিভিন্ন Event বা ঘটনা ঘটার সাথে কিছু নতুন কার্য্যকলাপ ঘটাতে হয়। যেমনঃ কোনো এলিমেন্ট এর ওপর মাউস রাখলে কি করবে, কোনো input field এ focus করুলে কি ঘটবে ইত্যাদি বলে দেওয়ার মাধ্যমই হলো Pseudo Class (স্যুডো ক্লাস)। কিছু স্যুডো ক্লাস এর নাম দেওয়া হলো।

কিছু Pseudo Class এর উদাহরণ

a:active
input:checked
input:disabled
p:empty
input:focus
button:hover
এমন আরো অনেক স্যুডো ক্লাস আহে চাইলে গুগল করে দেখে নিতে পারো।
